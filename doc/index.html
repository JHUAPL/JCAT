The Java CRISM Analysis Tool (JCAT) can be used to view CRISM data downloaded from the Planetary Data System (PDS) Geosciences Node at Washington University in St. Louis.

JCAT requires Java version 8 or later to run, along with Java FX. The Java SE Runtime Environment 8 (JRE) from Oracle includes both.

JCAT is distributed as a jar file. Version 1.0 was released on Nov 1, 2019. Double click the jar file, or from the command line, type 

    java -jar JCAT.jar

Follow the JCAT Tutorial to get started.
